## GitHub Projects as a Kanban board

Use GitHub Projects as a Kanban board — Track applications with columns like Researching → Applied → Phone Screen → Interview → Offer → Closed. This is genuinely useful and also shows process thinking if anyone sees it.

## Use GitHub Issues as a CRM

Log each company as an Issue with labels (recruiter-reach-out, applied, referral, etc.) and due dates for follow-ups. Lightweight but powerful.
